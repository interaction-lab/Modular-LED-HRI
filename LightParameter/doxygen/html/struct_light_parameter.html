Parameters
PatternThis variable holds one of the supported patterns
dirSpecifies the direction in which the pattern will run
startSpecifies the start time of the parameter
CycleSpecifies the number of times the pattern is supposed to run
IndexSpecifies the Index from which the pattern is suppose to start from
onThis variable is Specifically used for only On_AND_OFF pattern. Specifies the time for which the LED's should be on.
offThis variable is Specifically used for only On_AND_OFF pattern. Specifies the time for which the LED's should be off.
BrightnessUsed to set the brightness of the LED's.
color1Used to set the color of the pattern(used by majority of the patterns).
color2Used to set the 2nd color of the pattern(in this library only THEATER_CHASE uses this parameter).
IntervalUsed to set the time for which 1 cycle will run
g[]Stores the ID's of group of LED's in a pattern.
